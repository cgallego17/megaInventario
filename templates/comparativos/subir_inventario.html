Plantilla de Ejemplo

Descarga la plantilla de ejemplo y úsala como referencia. Tu archivo debe tener un formato similar a este:

codigo_barras codigo nombre atributo cantidad_sistema1 cantidad_sistema2
1234567890123 PROD001 Producto Ejemplo 1 Ejemplo (llenar) (llenar)
9876543210987 PROD002 Producto Ejemplo 2 Ejemplo (llenar) (llenar)
5555555555555 PROD003 Producto Ejemplo 3 Ejemplo (llenar) (llenar)
Importante: La plantilla incluye TODOS los productos del sistema.
  • Opción 1: Complete ambas columnas (cantidad_sistema1 y cantidad_sistema2) en el mismo archivo y súbalo dos veces (una para cada sistema).
  • Opción 2: Cree archivos separados - copie la plantilla, elimine una columna y renombre la otra como cantidad para cada sistema.
Las columnas codigo_barras, codigo, nombre y atributo son informativas y NO deben modificarse.
Información sobre el formato:
  • La plantilla incluye TODOS los productos del sistema - Complete las cantidades según corresponda
  • Nombres de columnas aceptados:
    • Para código: codigo_barras, codigo, barcode, codigo de barras
    • Para cantidad: cantidad, cantidad_sistema1, cantidad_sistema2, stock, stock_actual, inventario
  • El archivo puede ser Excel (.xlsx, .xls) o CSV
  • La primera fila debe contener los nombres de las columnas
  • El sistema comparará todos los productos que estén en la plantilla con el inventario físico
  • Si un producto no tiene cantidad (0 o vacío), también se comparará
  • Recomendación: Use la misma plantilla para ambos sistemas, solo complete la columna correspondiente a cada sistema
